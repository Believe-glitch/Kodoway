Flexbox : aligner ses éléments

Flexbox organise des éléments en ligne ou en colonne, et les aligne facilement. C'est ce que Kodoway utilise pour son propre menu !

Les propriétés de base

  • display: flex; active Flexbox sur un conteneur.
  • justify-content aligne les éléments horizontalement.
  • align-items aligne les éléments verticalement.
  • gap ajoute un espace régulier entre les éléments.

Exemple

style.css
1.conteneur { 2 display: flex; 3 justify-content: center; 4 gap: 16px; 5}
Aperçu — les 3 cases passent en ligne

Dès que tu mets display: flex; sur un conteneur, tous ses enfants directs se placent automatiquement en ligne.
