Les fonctions

Une fonction est un bloc de code réutilisable. Au lieu de réécrire le même code plusieurs fois, on l'enferme dans une fonction qu'on peut appeler autant de fois qu'on veut.

Déclarer une fonction

  • function nom() { } déclare une fonction.
  • Les éléments entre parenthèses sont les paramètres (les informations qu'on lui donne).
  • return renvoie un résultat depuis la fonction.
  • On appelle une fonction avec son nom suivi de parenthèses : nom().

Exemple

script.js
1function addition(a, b) { 2 return a + b; 3} 4console.log(addition(2, 3)); // affiche 5
Aperçu — ce que console.log() affiche

Ici, addition prend deux paramètres (a et b) et renvoie leur somme grâce à return.
