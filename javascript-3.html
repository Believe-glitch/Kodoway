Les événements

Un événement, c'est une action de l'utilisateur : un clic, une touche pressée, une page qui se charge. Le JavaScript permet de réagir à ces événements.

Réagir à un clic

  • document.querySelector('.classe') sélectionne un élément de la page.
  • .addEventListener('click', fonction) écoute un clic sur cet élément.
  • 'click' est l'événement le plus courant, mais il en existe d'autres ('input', 'submit'...).

Exemple

script.js
1const bouton = document.querySelector('.mon-bouton'); 2bouton.addEventListener('click', function() { 3 document.querySelector('#msg').textContent = 'Tu as cliqué !'; 4});
