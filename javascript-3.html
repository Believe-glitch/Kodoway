Les événements

Un événement, c'est une action de l'utilisateur : un clic, une touche pressée... Le JavaScript permet de réagir à ces actions.

Réagir à un clic

  • document.querySelector('.classe') sélectionne un élément de la page.
  • .addEventListener('click', fonction) écoute un clic sur cet élément.
  • 'click' est l'événement le plus courant.

Exemple

script.js
1const monBouton = document.querySelector('.mon-bouton'); 2monBouton.addEventListener('click', function() { 3 alert('Tu as cliqué !'); 4});
