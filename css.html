Qu'est-ce que le CSS ?

Le CSS sert à habiller le HTML : couleurs, polices, espacements, mise en page. Sans CSS, un site n'a ni couleur ni style.

Lier le CSS à une page HTML

  • <link rel="stylesheet" href="style.css"> relie le fichier CSS.
  • Un sélecteur cible les éléments à styliser.
  • p { } sélectionne tous les paragraphes (sélecteur d'élément).
  • .classe { } sélectionne tous les éléments ayant cette classe.

Exemple

style.css
1.titre { 2 color: blue; 3}
Aperçu — la règle appliquée à un vrai élément

Cette règle dit : "l'élément qui a la classe titre doit être bleu". Une règle CSS, c'est toujours : un sélecteur, puis des accolades { } contenant des propriétés.
