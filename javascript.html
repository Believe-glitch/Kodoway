Qu'est-ce que le JavaScript ?

Le JavaScript donne vie à tes pages : boutons qui réagissent, contenus qui changent, animations.

Les variables

  • let déclare une variable qui peut changer de valeur.
  • const déclare une variable qui ne change jamais (constante).
  • console.log() affiche une information dans la console.

Exemple

script.js
1const prenom = "Léa"; 2console.log(prenom);
Aperçu — ce que console.log() affiche

En vrai, ce message s'affiche dans la console du navigateur (touche F12). Ici, Kodoway te la montre directement sous le code.
