Le responsive design

Ton site doit s'afficher correctement sur ordinateur, tablette et mobile, grâce aux media queries.

Les media queries

Une media query applique des règles CSS seulement si une condition est vraie, par exemple "si l'écran fait moins de 600px de large".

  • @media (max-width: 600px) { } cible les petits écrans.
  • Les règles à l'intérieur écrasent les règles normales sur mobile.

Exemple

L'aperçu ci-dessous est volontairement étroit (comme un téléphone) pour que tu voies l'effet de la media query :

style.css
1@media (max-width: 600px) { 2 .carte { 3 flex-direction: column; 4 } 5}

C'est exactement cette technique que Kodoway utilise pour transformer son menu en liste déroulante sur mobile. Bravo, tu termines le cours CSS !
