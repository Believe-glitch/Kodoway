Cours JavaScript · Aide-mémoire

Tout le vocabulaire JavaScript

Un aide-mémoire avec les mots-clés et concepts les plus utiles en JavaScript, leur rôle, et quelques cas particuliers à connaître. Reviens ici quand tu en as besoin.

Variables et types

let nom = valeur;

Déclare une variable qui peut changer de valeur plus tard.

const nom = valeur;

Déclare une constante : sa valeur ne peut plus changer.

⚠️ Cas particulier : préfère toujours const, et utilise let seulement si tu sais que la valeur va changer.
"texte"

Une chaîne de caractères (string), entre guillemets simples ou doubles.

42

Un nombre (number), sans guillemets.

true / false

Un booléen (boolean) : vrai ou faux.

[1, 2, 3]

Un tableau (array) : une liste ordonnée de valeurs.

Opérateurs

+ - * /

Addition, soustraction, multiplication, division.

===

Égalité stricte : vérifie si deux valeurs sont identiques (valeur et type).

⚠️ Cas particulier : évite == (sans le troisième égal), il peut donner des résultats surprenants.
!==

Différence stricte : vrai si les deux valeurs sont différentes.

> < >= <=

Comparaisons : plus grand, plus petit, plus grand ou égal, plus petit ou égal.

&& / ||

ET logique (les deux conditions vraies) / OU logique (au moins une vraie).

Fonctions

function nom(param) { }

Déclare une fonction réutilisable, avec des paramètres entre parenthèses.

return valeur;

Renvoie un résultat depuis la fonction, et arrête son exécution.

nom(argument);

Appelle (exécute) la fonction avec une vraie valeur.

const f = () => { };

Une fonction fléchée : une autre façon d'écrire une fonction, plus courte.

Conditions et boucles

if (condition) { }

Exécute le bloc seulement si la condition est vraie.

else if (...) { }

Teste une autre condition si la précédente était fausse.

else { }

Exécute ce bloc si aucune condition précédente n'était vraie.

for (let i = 0; i < 5; i++) { }

Répète un bloc de code un nombre précis de fois.

⚠️ Cas particulier : attention aux boucles infinies si la condition n'est jamais fausse.
while (condition) { }

Répète un bloc tant que la condition reste vraie.

Manipuler le DOM

document.querySelector('.classe')

Sélectionne le premier élément correspondant au sélecteur CSS donné.

document.querySelectorAll('.classe')

Sélectionne tous les éléments correspondants (sous forme de liste).

.textContent

Lit ou modifie le texte d'un élément.

.classList.add('x')

Ajoute une classe CSS à un élément.

.classList.remove('x')

Retire une classe CSS d'un élément.

.classList.toggle('x')

Ajoute la classe si absente, la retire si présente.

Les événements

.addEventListener('click', fn)

Exécute la fonction fn quand l'élément est cliqué.

'input'

Événement déclenché à chaque frappe dans un champ de texte.

'submit'

Événement déclenché à l'envoi d'un formulaire.

⚠️ Cas particulier : pense à appeler event.preventDefault() pour empêcher le rechargement de la page.
console.log(valeur)

Affiche une valeur dans la console du navigateur, pratique pour vérifier ton code.
