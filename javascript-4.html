Manipuler le DOM

Le DOM (Document Object Model), c'est la représentation de ta page que le JavaScript peut lire et modifier : changer un texte, ajouter une classe CSS, etc.

Lire et modifier la page

  • document.querySelector('h1') sélectionne le premier élément qui correspond.
  • .textContent lit ou change le texte d'un élément.
  • .classList.add('classe') ajoute une classe CSS à un élément.
  • .classList.toggle('classe') ajoute la classe si absente, la retire si présente.

Exemple

script.js
1document.querySelector('h1').textContent = "Bonjour";
Aperçu — le titre change réellement

Cette ligne va chercher le premier <h1> de la page et remplace son texte par "Bonjour" — instantanément, sans recharger la page.
