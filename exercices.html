Les défis Kodoway

Ici, pas de leçon pour te guider : ce sont de vrais défis, du plus facile au plus difficile. Certains combinent plusieurs notions à la fois (HTML + CSS, ou HTML + JavaScript) — normal, c'est voulu ! Si tu as fait les 3 cours, tu as tout ce qu'il faut.

🧩 Défi 1 Facile

Crée une mini carte de présentation : un titre <h2> avec le texte Mon profil, et un paragraphe <p> avec le texte Je code avec Kodoway. (L'ordre n'a pas d'importance.)

Aperçu de ton code

🧩 Défi 2 Facile

Écris une seule règle CSS pour la classe .carte qui combine deux propriétés à la fois : display: flex; ET padding: 20px;.

Aperçu de ton code

🧩 Défi 3 Moyen

Mélange HTML + CSS : écris un <nav> contenant 3 liens <a> (Accueil, Cours, Contact), puis une balise <style> qui transforme ce nav en ligne flexible avec display: flex;.

Aperçu de ton code

🧩 Défi 4 Moyen

Écris une fonction estMajeur(age) qui renvoie true si age est supérieur ou égal à 18, et false sinon.

Aperçu de ton code

🧩 Défi 5 — Le grand défi Difficile

Tout en un : crée un <button> et un <p>, puis utilise JavaScript pour qu'un clic sur le bouton change le texte du paragraphe pour qu'il contienne le mot Bravo. (HTML + CSS si tu veux + JavaScript, à toi de tout assembler.)
