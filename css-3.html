Le modèle de boîte

Chaque élément HTML est une boîte rectangulaire. Le CSS permet de contrôler l'espace autour et à l'intérieur de cette boîte.

Les couches d'une boîte

  • padding : l'espace à l'intérieur de la boîte.
  • border : la bordure de la boîte.
  • margin : l'espace à l'extérieur de la boîte.
  • width / height : la largeur et la hauteur.

Exemple

style.css
1.carte { 2 padding: 20px; 3 border: 1px solid gray; 4 margin: 10px; 5}
Aperçu

Astuce : le padding est le passe-partout, le border est le cadre, et le margin est l'espace sur le mur autour du cadre.
