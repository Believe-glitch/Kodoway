Toutes les propriétés CSS utiles

Tu as fini la leçon 1 ? Voici un aide-mémoire avec les sélecteurs et propriétés les plus utilisées, leur rôle, et quelques cas particuliers. Reviens ici quand tu en as besoin.

Les sélecteurs

p { }

Sélecteur d'élément : cible toutes les balises p.

.classe { }

Sélecteur de classe : cible tous les éléments avec class="classe".

⚠️ Cas particulier : une classe peut être utilisée sur plusieurs éléments différents.
#id { }

Sélecteur d'id : cible l'unique élément avec id="id".

⚠️ Cas particulier : un id ne doit apparaître qu'une seule fois dans toute la page.
* { }

Sélecteur universel : cible absolument tous les éléments.

div p { }

Cible les p qui sont à l'intérieur d'un div (sélecteur descendant).

a:hover { }

Pseudo-classe : applique le style seulement quand la souris survole l'élément.

⚠️ Cas particuliers utiles : :first-child, :last-child, :focus (au clic dans un champ), :nth-child(2).

Texte et typographie

color

Couleur du texte.

font-size

Taille du texte (ex : 16px, 1.2rem).

font-weight

Épaisseur du texte (normal, bold, ou un nombre comme 600).

font-family

La police utilisée (ex : 'Inter', sans-serif).

text-align

Alignement du texte : left, center, right, justify.

text-decoration

Souligner, barrer ou retirer le soulignement (underline, line-through, none).

⚠️ Cas particulier : text-decoration: none; retire le soulignement des liens <a>.
line-height

Hauteur de ligne, pour l'espacement vertical entre les lignes de texte.

Couleurs et fond

background-color

Couleur de fond d'un élément.

background-image

Image de fond : url('chemin.jpg').

opacity

Transparence d'un élément, de 0 (invisible) à 1 (opaque).

Le modèle de boîte

width / height

Largeur et hauteur d'un élément.

padding

Espace intérieur, entre le contenu et la bordure.

margin

Espace extérieur, entre la boîte et ses voisins.

⚠️ Cas particulier : margin: 0 auto; centre horizontalement un bloc qui a une largeur définie.
border

Bordure : épaisseur, style et couleur (ex : 1px solid gray).

border-radius

Arrondit les coins d'un élément.

box-sizing: border-box

Fait en sorte que padding et border soient inclus dans width, pour des calculs plus simples.

Affichage et positionnement

display

Comment l'élément se comporte : block, inline, flex, grid, none (caché).

⚠️ Cas particulier : display: none; cache complètement l'élément (il ne prend plus de place).
position

static (par défaut), relative, absolute ou fixed (reste à l'écran même en scrollant).

top / left / right / bottom

Décale un élément quand position n'est pas static.

z-index

Ordre d'empilement : un nombre plus grand passe au-dessus.

Flexbox

display: flex

Active Flexbox : les enfants directs se placent en ligne.

flex-direction

row (ligne, par défaut) ou column (colonne).

justify-content

Alignement horizontal : center, space-between, flex-end...

align-items

Alignement vertical : center, flex-start, stretch...

gap

Espace régulier entre les enfants, sans avoir besoin de margin.

flex-wrap: wrap

Permet aux éléments de passer à la ligne suivante s'ils manquent de place.

Responsive

@media (max-width: 600px) { }

Applique des règles seulement sous une certaine largeur d'écran.

% / vw / vh

Unités relatives à la taille du parent ou de l'écran (viewport width/height).

rem / em

Unités relatives à la taille de police (rem = taille racine, em = taille du parent).

⚠️ Cas particulier : préfère rem pour font-size, ça respecte les préférences d'accessibilité du visiteur.
