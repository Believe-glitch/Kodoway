Les formulaires

Un formulaire permet à un visiteur d'envoyer des informations : son nom, son message, etc. C'est la base de tout formulaire de contact.

Les éléments d'un formulaire

Un formulaire commence par la balise <form>. À l'intérieur, on place des champs avec <input> et un bouton pour l'envoyer.

  • <input type="text"> crée un champ de texte.
  • <label> décrit un champ pour l'utilisateur.
  • placeholder="..." affiche un texte d'exemple grisé.
  • <button type="submit"> envoie le formulaire.

Exemple

exemple.html
1<label>Ton prénom</label> 2<input type="text" placeholder="Ex : Léa"> 3<button type="submit">Envoyer</button>
Aperçu

On verra plus tard avec le CSS comment rendre ces champs beaux, et avec le JavaScript comment réagir quand on clique sur "Envoyer".
