Les conditions

Une condition permet à ton code de faire un choix : "si telle chose est vraie, fais ceci, sinon fais autre chose." C'est la base de toute logique en programmation.

if / else

  • if (condition) { } exécute le bloc si la condition est vraie.
  • else { } exécute un autre bloc si la condition est fausse.
  • === vérifie si deux valeurs sont égales.
  • >, <, >=, <= comparent des nombres.

Exemple

script.js
1const age = 20; 2if (age >= 18) { 3 console.log("Majeur"); 4} else { 5 console.log("Mineur"); 6}
